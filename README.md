# 📝 Todo List App

A simple and user-friendly Todo List application built to manage daily tasks efficiently. This project allows users to add, update, and delete tasks easily.

## 🚀 Features
- ➕ Add new tasks
- ✏️ Edit existing tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 📋 Simple and easy-to-use interface

## 🛠️ Technologies Used
- Python
- (Add Tkinter / Flask / HTML-CSS-JS if used)

## 📌 Project Description
The Todo List App helps users organize their daily activities by creating and managing task lists. It provides a simple interface to track completed and pending tasks, improving productivity and time management.

## 💡 Future Improvements
- Add task deadlines
- Add reminders/notifications
- Save tasks permanently using database
- Dark mode feature
